ForcePowers

Number of arguments: 13

Example:

ForcePowers		4000.0, 0.0, 1.5, 1.0, 120.0, 0.0, 6.0, 0.35, 0.70, 1.00, 7000.0, 9000.0, 1000.0

The AI will trigger a character's COG to produce Force Power-like effect on different time intervals.

Arguments:

Argument # Argument meaning
1 Interval in milliseconds to check for possible FPs
2 Min dist. from target to use FP 1
3 Max dist. from target to use FP 1
4 Min dist. from target to use FP 2
5 Max dist. from target to use FP 2
6 Min dist. from target to use FP 3
7 Max dist. from target to use FP 3
8 % on a given interval to use FP1 (0=never, 1=always)
9 % on a given interval to use FP2 (checked only if prev. percentage fails)
10 % on a given interval to use FP3 (checked only if prev. percentage fails)
11 Recharge time for FP1 (msec)
12 Recharge time for FP2 (msec)
13 Recharge time for FP3 (msec)
