Note for Warp. Jk units are very big so you don`t need to change the Cordinates very much To warp. Suggested use is to get your "coords", then turn of the map rotation if on. decide which direction you want to go. For instance coords = -1.97,-3.05,-24.181 and you want to go right or left use something like -1.5 -3.05 -24.181 and see how far you move and which direction.
