LobFire

Number of arguments: 6

Example:

LobFire         	2500.0, 0.8, 0.5, 4.0, 0.20, 0

This instinct is used at characters throwing projectiles like ReeYees.

Arguments:

Argument # Argument meaning
1 Time interval to check for throw (msec)
2 Min Dot: 1.0=must be facing exactly, 0.0=180 degrees arc, -1.0=full 360 degrees arc.
3 Minimal distance to target to be able to throw TDs
4 Maximal distance to target to be able to throw TDs
5 Error. Ranges from 0 (perfect aim) to 1 (random)
6 PercentPrimary: if 1- use only primary weapon, if 0-this one.
